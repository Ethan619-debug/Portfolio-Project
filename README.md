# My Portfolio

A modern, responsive portfolio website built with Tailwind CSS.

## Project Structure

```
Portfolio/
│
├── index.html              # Main landing page
├── README.md               # Project documentation
│
├── css/
│   └── style.css          # Custom CSS styles
│
├── js/
│   └── script.js          # JavaScript functionality
│
├── assets/
│   ├── images/            # Store project screenshots, profile images
│   └── icons/             # Store SVG icons, favicon
│
└── pages/                 # Additional pages (future use)
    ├── blog.html          # Blog page template
    └── projects.html      # Detailed projects page template
```

## Features

- ✨ Modern gradient design with blue-to-purple theme
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Built with Tailwind CSS
- ⚡ Smooth animations and transitions
- 🎯 Easy to customize and extend

## Getting Started

1. Open `index.html` in your browser
2. Customize your content:
   - Update your name in the hero section
   - Add your projects with descriptions
   - Update contact information
   - Add profile image in `assets/images/`

## How to Use Folders

- **css/** - Place all custom stylesheets here
- **js/** - Store JavaScript files for functionality
- **assets/images/** - Add portfolio images, screenshots, and profile photos
- **assets/icons/** - Store SVG icons for social media or features
- **pages/** - Future pages like detailed project pages, blog, etc.

## Customization Tips

- Change colors in Tailwind classes (e.g., `from-blue-500` to `from-green-500`)
- Update text and content directly in `index.html`
- Add more project cards by duplicating the project-card div
- Add new navigation links in the navbar

## Future Enhancements

- Add a blog section in `pages/blog.html`
- Create detailed project pages in `pages/projects.html`
- Add a contact form
- Integrate with a CMS
- Portfolio filtering by category
